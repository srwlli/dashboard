## Component Reuse Strategy

**Components to Use (Already Exist):**
- `PageLayout` - Main layout wrapper with header
- `Sidebar` - Navigation sidebar with active state
- Tab navigation - Can use existing button/tab patterns
- Filter dropdowns - Use existing form components
- Card components - Existing card styling for workorder/stub items

**New Components to Create:**
- `WorkorderCard` - Displays single workorder summary
- `StubCard` - Displays single stub summary
- `TabNavigation` - Tab switcher for Workorders/Stubs/Docs
- `FilterBar` - Filter dropdown interface

**Hooks to Create:**
- `useWorkorders()` - Fetch and cache workorders from `/api/workorders`
- `useStubs()` - Fetch and cache stubs from `/api/stubs`
- `useWorkorderDetail(id)` - Fetch specific workorder from `/api/workorders/:id`
